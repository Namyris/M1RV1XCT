# Contributing

Merci de contribuer a ce projet. Voici les etapes a suivre.

## Processus

1. Fork le repo
2. Creez une branche (`git checkout -b feature/votre-ajout`)
3. Commitez (`git commit -m 'Description de la modification'`)
4. Push (`git push origin feature/votre-ajout`)
5. Ouvrez une Pull Request

## Regles

- HTML/CSS vanilla uniquement
- Responsive obligatoire
- Images optimisees (max 150KB)
- Code lisible et commente si necessaire

## Signaler un probleme

Ouvrez une issue en precisant le navigateur, l'OS et des captures d'ecran.
