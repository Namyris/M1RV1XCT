Portfolio minimaliste et laboratoire creatif. Code source du site personnel heberge sur GitHub Pages.

## Apercu

Site vitrine responsive presentant mes projets, competences et parcours dans le developpement web et le design d'interfaces. Concu pour etre leger, rapide et entierement accessible.

## Tech Stack

- **HTML5** - Balisage semantique
- **CSS3** - Variables CSS, Grid, Flexbox, animations
- **GitHub Pages** - Deploiement automatique
- **Google Fonts** - Typographie (Inter, Poppins)

## Utilisation locale

```bash
git clone https://github.com/Namyris/M1RV1XCT.git
cd M1RV1XCT
python -m http.server 8000
```
